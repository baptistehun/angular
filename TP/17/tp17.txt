Cordova
=======
1/ Installer cordova : npm install -g cordova
2/ taper cordova -v
3/ Ouvrir index.html
4/ Ajouter la ligne : <script type=”text/javascript” src=”cordova.js”></script>
5/ Effectuer un ng build
6/ Installer la plateforme Android : cordova platform add android
7/ Depuis la racine du répertoire TP/ créer un projet Corvoda
    cordova create RVGCordova com.cyrilvincent RVG
8/ Depuis RVGCordova installer la plateforme Android
    cordova platform add android
9/ Faire de même pour iOS : cordova platform add iOS
10/ Faire de même pour browser : cordova platform add browser
    Pour info Vidual Studio Code est programmé avec ce mode
11/ Faire de même pour Windows : cordova platform add windows
12/ Exécuter la plateforme browser : cordova run browser
13/ Copier le contenu de RVG/dist/RVG15/* vers RVGCordova/www/
13/ cordova run browser
14/ Android
    Prérequis Android SDK configuré, Java 8 64 bits, HAXM
    set ANDROID_HOME=D:\Programmes\Android\SDK
    set JAVA_HOME=C:\Program Files\Java\jdk1.8.0_131
    cordova run android
    Il est posside de définir l'AVD : cordova emulate --target=Android22 android
15/ Récupérer le .apk dans platforms\android\app\build\outputs\apk\debug et tester sur un smartphone
16/ Tester cordova run windows
    Prérequis : .NET SDK + Windows 10
17/ Tester cordova build iOS
    Nécessite un Mac pour compiler
