Cordova
=======
1/ Installer cordova : npm install -g cordova
2/ taper cordova -v
3/ Installer la plateforme Android : cordova platform add android
4/ Depuis la racine du répertoire TP/ créer un projet Corvoda
    cordova create RVG-cordova com.cyrilvincent RVG
5/ Depuis RVG-cordova installer la plateforme Android
    cordova platform add android
6/ Faire de même pour iOS : cordova platform add iOS
7/ Faire de même pour browser : cordova platform add browser
8/ Faire de même pour Windows : cordova platform add windows
    Pour info Vidual Studio Code est programmé avec ce mode
9/ Exécuter la plateforme browser : cordova run browser
    Il est possible de modifier le fichier RVGCordova/www/index.html
10/ Android
    Prérequis Android SDK configuré, Java 8 64 bits, HAXM
    set ANDROID_HOME=D:\Programmes\Android\SDK
    set JAVA_HOME=C:\Program Files\Java\jdk1.8.0_131
    cordova run android
11/ Récupérer le .apk dans platforms\android\app\build\outputs\apk\debug et tester sur un smartphone
12/ Tester cordova run windows
    Prérequis : .NET SDK + Windows 10
13/ Tester cordova build ios
    Nécessite un Mac pour compiler
14/ Copier le contenu de RVG/dist/RVG/* vers RVG-cordova/www/
15/ Ouvrir index.html
16/ Ajouter la ligne : <script type=”text/javascript” src=”cordova.js”></script>
17/ Modifier la la ligne <base href="."> (peut être effectué avec ng build)
18/ Tester sur Android
19/ Installer le sur votre téléphone
